## v0.6 · 2026-05-17

### 修改

- 技术栈中的 Phaser 版本从 Phaser 3 调整为 Phaser 4.1.x。
- 当前工程锁定 `phaser@4.1.0`，用于 Vite + React 项目的战斗场景渲染层。

### 原因

- 项目是从零开始的新工程，没有 Phaser 3 历史代码包袱。
- 战斗核心逻辑独立在 `src/core`，Phaser 只承载渲染与动效，版本风险被限制在表现层。
- Phaser 4.1.x 对现代 ESM / Vite 工程更友好，并更适合后续卷轴、朱砂、符火、印章、滤镜等 2D 表现需求。

### 保留

- React / Phaser 与核心战斗逻辑继续分离。
- `src/core` 不依赖 React、Phaser 或 DOM。
- 第一阶段仍只做浏览器内核心战斗 MVP，不提前做桌面打包。
