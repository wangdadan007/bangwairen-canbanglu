## v2.4 · 2026-05-22

### 来源

- T65 执行第一章三角色路线 / 奖励 / 法宝构筑倾向补强。
- 用户要求补齐此前评估中“中间三点”未完全达标的问题：路线选择可读性、角色长期奖励差异、法宝作为构筑锚点，以及 Boss 对路线倾向的轻量反馈。

### 修改

- 第一章路线页不再只显示节点类型与状态，新增路线倾向标签和可达节点提示，用于说明稳行、归册、裂榜、补给、高压等选择会把本局推向什么方向。
- 战后奖励 offer 增加角色倾向排序：衡简偏归册 / 朱批 / 抽牌，照微偏问名 / 断异动 / 辨势，莲烬偏破形 / 香火 / 劫数与榜裂。
- 第一章途中和 Boss 后法宝 offer 保持三选一，但候选排序会根据角色和已走路线倾向调整，法宝页同步显示构筑锚点标签。
- 窃榜使 Boss 增加轻量路线响应：归册倾向优先遮名开场，裂榜或高压倾向优先塞污卷开场，其余路线保持原来按册来势开场。

### 原因

- T64 已解决路线可达和三主线结构，但还需要让玩家感到路线、角色和法宝选择在中长期构筑上互相回应。
- 三角色不能只停留在初始牌组和起始法宝差异；奖励与法宝 offer 也需要让玩家看到自己正在形成某种 build。
- Boss 对路线的反馈应先做轻量、可控、可测试版本，避免第一章直接变成强惩罚或角色锁路。

### 保留

- 不新增第 4 角色，不新增第二章，不新增 Boss 池，不做完整随机地图。
- 不改变三角色固定起始法宝 + 途中三选一 + Boss 后三选一的节奏。
- 不改变形、问名、正名、伏诛 / 归册、裁定三类、法宝不入牌堆等核心规则。
- 路线仍然只提供风格倾向，不做角色专属路线或隐性锁路。

## v2.3 · 2026-05-21

### 来源

- T64 开发落地第一章三主线路线分支。
- T64 测试发现裂榜线可能绕开旧的“窃香吏”中段法宝触发点，导致部分路线缺少第一章途中法宝补给。

### 修改

- 第一章路线数据正式改为前三场固定教学，第四节点后可选稳行、归册、裂榜三条主线，每条主线中段各有一次二选一小分支，并最终汇回窃榜使 Boss。
- 路线状态增加路线倾向记录：`steady`、`catalogue`、`fracture`、`supply`、`high_pressure`，用于后续 Boss 变体、结算摘要和 QA 观察。
- 路线页从只展示线性节点改为可选择多个可达下一节点；选择后 run / route 战斗序列同步。
- 第一章途中法宝三选一从只绑定 `encounter_elite_incense_clerk`，扩展为“旧遭遇 checkpoint 或完成 5 场遭遇”双条件，保证绕开窃香吏的路线仍有中段法宝补给。

### 原因

- 分支路线需要让玩家在第 4 节点后主动规划风险、补给和构筑方向，而不是继续被 `nextNodeIds[0]` 线性推进。
- 法宝途中补给是三角色第一章节奏的一部分，不能因选择裂榜线而丢失。

### 保留

- 当前 Boss 仍为单一窃榜使遭遇，本轮只记录路线倾向，不按路线直接强化 Boss。
- 不做完整随机地图、不新增章节、不新增 Boss 池、不新增生产依赖。

## v2.2 · 2026-05-21

### 来源

- 用户确认三条主路线即使与三个角色风格适配，也只能是倾向，不能让某个角色只能走某一条路线。
- 用户要求确保每个角色都能玩任意一条主线和支线，并且都能以合理的己形通关第一章 Boss。

### 修改

- T64 增加“软匹配、禁锁路”原则：稳健修榜线、归册精英线、榜裂削籍线可以分别更适合衡简、照微、莲烬，但不得出现角色限制、角色专属路线或隐性锁路。
- T64 验收增加 3 角色 × 3 主线矩阵验证，并要求每条主线的两条中段支线都至少被一个非推荐角色覆盖。
- T64 QA 记录增加 Boss 前己形、Boss 后己形、路线倾向和补给使用记录；若某组合只能靠极低己形硬过，必须调整节点补给、敌人压力、奖励或 Boss 倾向。
- 高压、高榜裂和高归册路线的 Boss 强化必须有上限，并保留休整、商店或等价补给抵消空间。

### 原因

- 肉鸽路线选择应该表达“这局想怎么玩”，不能把角色选择变成固定路线答案。
- 三角色的差异应通过初始牌组、起始法宝、风险曲线和奖励偏好形成风格倾向，而不是通过路线可达性限制。
- 第一章公开 Demo 需要可复玩性和公平感；任意角色走任意主线都应能形成合理通关路径。

### 保留

- 三条主线仍保留风格倾向，不取消稳健、归册、榜裂 / 削籍三种路线身份。
- 不改变三角色己形梯度、固定起始法宝或法宝获得节奏。
- 不新增第 4 角色，不新增章节，不新增 Boss 池。
- 不做完整随机地图，不改变核心玩法术语和裁定规则。

## v1.9 · 2026-05-21

### 来源

- 用户进一步判断莲烬作为直攻冲阵角色应当己形最高，衡简第二，照微最低。
- 用户追问照微形最低时，前期需要什么特点支撑。

### 修改

- 三角色己形梯度调整为：莲烬 78 / 78，衡简 72 / 72，照微 64 / 64。
- 照微的前期支撑明确为：固定起始法宝照骨镜、更多问名 / 点额、提前辨势、首战提示，以及更容易在前两场打出正名、名破和高质量奖励。
- 莲烬的高己形定义为直攻缓冲，允许吃来势换快节奏；但躁火反噬仍保留，避免高己形变成无脑莽。

### 原因

- 莲烬是冲阵直攻角色，最高己形符合火性、莲骨、以身破局的气质，也能让新玩家获得直接爽感。
- 衡简是默认稳健角色，不需要最高己形；他的稳定性来自平衡牌组和打神鞭残节把问名收益转成收尾。
- 照微可以最低形，但必须靠信息优势和早期正名收益形成另一种安全感。

### 保留

- 不改变三角色固定起始法宝：衡简打神鞭残节，照微照骨镜，莲烬赤绫火轮。
- 不改变形、问名、正名、伏诛 / 归册、裁定三类等核心规则。
- 本次只更新 PRD 和计划口径，不实现数值代码。
