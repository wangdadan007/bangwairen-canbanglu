# PRD-content-scope：第一章内容范围

版本：v0.5  
日期：2026-05-16  
建议路径：`docs/prd/PRD-content-scope.md`

## 1. 第一章定位

第一版 Demo 只做第一章。

章节名：残榜初裂。

目标时长：单局约 30 到 45 分钟。

第一章的目标不是展示所有正式版内容，而是让玩家完整理解：形、名、问名、正名、伏诛、归册、裁定、来势、异动、封势、朱批、三坛、法宝认主、反噬、劫数、榜裂。

## 2. 内容规模

| 模块 | 开发 MVP | 公开 Demo 目标 | 备注 |
|---|---:|---:|---|
| 可玩角色 | 1 | 1 | 执簿者 |
| 卡牌 | 约 40 | 42-48 | 先保证五条构筑线闭合 |
| 普通敌人单位 | 8 | 12 | 公开 Demo 需要更像肉鸽池子 |
| 低成本杂祟 / 召唤物 | 2-3 | 4 | 可复用美术和简化名机制 |
| 普通遭遇模板 | 10-12 | 18-22 | 优先扩模板，不盲目堆美术 |
| 精英敌人 | 3 | 3 | 保持 3 个精英候选 |
| 事件精英 | 0 | 可选 1 | 高榜裂或事件失败时出现 |
| Boss | 1 | 1 | 窃榜使 |
| Boss 行为变体 | 1 | 2-3 | 根据榜裂、归册、登簿倾向变化 |
| 法宝 | 8 | 10-12 | MVP 先做 8 件 |
| 事件 | 6 | 8-10 | 6 个偏少，公开 Demo 容易重复 |
| 路线地图 | 简版 | 完整第一章路线 | 节点像榜面名籍 |
| 商店 / 休整 / 结算页 | 简版 | 完整可用 | 结算突出伏诛、归册、裁定记录 |

## 3. 第一章普通敌人单位

| 敌人 | 类型 | 定位 | 主要机制 |
|---|---|---|---|
| 纸面鬼 | 有名普通 | 教学单怪 | 形、名、问名、正名 |
| 无名纸祟 | 无名杂祟 | 弱杂祟 | 无名、低形、多敌组合填充 |
| 偷香鼠 | 有名普通 | 资源干扰 | 偷香、异动、断供教学 |
| 香灰虱 | 无名杂祟 | 弱群怪 | 群体压力，配合偷香鼠 |
| 铜铃夜巡 | 有名普通 | 来势教学 | 稳定来势、封势教学 |
| 无灯庙祝 | 有名普通 | 支援怪 | 祈愿、召唤、强化杂祟 |
| 断签客 | 有名普通 | 劫数教学 | 给玩家短期诱惑，埋劫数 |
| 灰坛童子 | 有名普通 | 地坛教学 | 敌行动前异动、坛位干扰 |
| 瘟纸人 | 有名普通 | 污染牌堆 | 塞污卷、反噬教学 |
| 塞卷吏 | 有名普通 | 牌堆污染 | 把污卷、残页塞进抽牌堆 |
| 逃名纸马 | 有名普通 | 遮名 / 逃亡 | 已问名后可能遮名或逃走 |
| 供桌残像 | 支援物 / 半敌人 | 支援 | 治疗、加香、护住主怪 |

无名纸祟、香灰虱、供桌残像可以使用低成本敌人规格，不需要完整问名链。

## 4. 精英与 Boss

| 敌人 | 类型 | 功能 |
|---|---|---|
| 窃香吏 | 精英 | 解锁墨，考验偷香、断供、问名收益 |
| 火部逃名者 | 精英 | 解锁削籍 / 榜裂，考验正名前后的机制差异 |
| 斗部空壳 | 精英 | 解锁天坛 / 法宝认主，考验三坛和法宝联动 |
| 旧榜残吏 | 事件精英，可选 | 只在榜裂较高或事件选择错误时出现 |
| 窃榜使 | Boss | 第一章综合考核，围绕偷名、遮名、伪归册、撕榜展开 |

Boss 行为变体：

| 变体 | 触发倾向 | 行为重点 |
|---|---|---|
| 低榜裂 | 榜裂低 | 偷名、遮名、伪归册 |
| 高榜裂 | 榜裂高 | 撕榜、塞污卷、法宝反噬 |
| 高归册 | 玩家归册多、朱批强 | 针对已登簿 / 已朱批内容做反制 |

Boss 名结构建议：榜下无名执役、窃取三部空缺、残榜裂源之一。

## 5. 普通遭遇模板

第一章不是每场固定一个敌人，而是使用“敌人单位 + 遭遇模板”的结构。

普通遭遇模板示例：

| 阶段 | 遭遇 | 目的 |
|---|---|---|
| 前段 | 纸面鬼 | 单怪教学 |
| 前段 | 纸面鬼 + 无名纸祟 | 教多目标但不复杂 |
| 前段 | 偷香鼠 | 教异动 |
| 前段 | 偷香鼠 + 香灰虱 | 教主怪 + 杂祟 |
| 前段 | 铜铃夜巡 | 教来势 / 封势 |
| 中段 | 无灯庙祝 + 无名纸祟 | 教召唤 / 支援 |
| 中段 | 断签客 | 教劫数诱惑 |
| 中段 | 塞卷吏 + 纸面鬼 | 教污卷干扰 |
| 中段 | 逃名纸马 + 香灰虱 | 教遮名和收尾压力 |
| 中段 | 灰坛童子 + 铜铃夜巡 | 教地坛时间点 |
| 后段 | 瘟纸人 | 教反噬 / 污染 |
| 后段 | 瘟纸人 + 塞卷吏 | 牌堆污染压力 |
| 后段 | 供桌残像 + 偷香鼠 | 支援怪保护异动怪 |
| 后段 | 灰坛童子 + 逃名纸马 | 问名与时机压力 |
| 后段 | 无灯庙祝 + 瘟纸人 | 召唤与污卷混合 |

公开 Demo 目标为 18 到 22 个普通遭遇模板。

## 6. 机制解锁顺序

第一章采用逐战斗机制解锁，第 4 场之后允许在已解锁机制范围内使用随机遭遇模板。

| 战斗 | 推荐敌人 / 类型 | 新增内容 |
|---|---|---|
| 1 | 纸面鬼 | 形、破形、问名、正名、名破、伏诛 / 归册 |
| 2 | 偷香鼠 | 来势、异动、封势、断供 |
| 3 | 铜铃夜巡 | 朱批 |
| 4 | 无灯庙祝 | 奉坛、人坛 |
| 5 | 窃香吏（精英） | 墨 |
| 6 | 断签客 / 中段普通遭遇 | 劫数 |
| 7 | 火部逃名者（精英） | 削籍、榜裂 |
| 8 | 灰坛童子 / 中后段普通遭遇 | 地坛 |
| 9 | 斗部空壳（精英） | 天坛、法宝认主 |
| 10 | 瘟纸人 / 后段普通遭遇 | 反噬 |
| Boss | 窃榜使 | 综合考核，不再新增基础系统 |

所有卡牌奖励池必须跟随机制解锁。未解锁机制不得出现在奖励、商店、事件、法宝描述中。

## 7. 初始牌组建议

角色：执簿者。  
初始生命：可先设 72，后续平衡调整。  
每回合香火：3。  
初始牌组：11 到 12 张。

| 数量 | 卡牌 | 费用 | 效果 |
|---:|---|---:|---|
| 4 | 朱符 | 1 | 破形 4。若目标已有名被揭示，额外破形 1 |
| 3 | 护案符 | 1 | 封势 3。只能降低直接来势 |
| 2 | 问名 | 1 | 揭示目标 1 格名。若目标无名可问，辨势并破形 1 |
| 1 | 点额 | 1 | 破形 2，并问名 1 |
| 1 | 整卷 | 0 | 查看抽牌堆顶 2 张，选择 1 张置底。若本回合完成正名，抽 1 张 |
| 1 | 静香 | 0 | 本回合下一次问名后，获得 1 香火。消耗 |

初始法宝建议：破笔。每场战斗第一次正名敌人时，获得 1 墨。

## 8. 事件与法宝规模

MVP 先做 8 件法宝、6 个事件。公开 Demo 前扩为 10 到 12 件法宝、8 到 10 个事件。

事件类型建议：给卡、删卡、朱批、榜裂诱惑、法宝认主、削籍代价、特殊战斗、商店折扣、休整变体、剧情判词。
