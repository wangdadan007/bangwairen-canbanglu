# 《榜外人：残榜录》素材来源与授权台账

日期：2026-05-18

## 1. 使用规则

本文件用于记录后续图片、音频、字体、视频、图标、生成式素材和第三方素材包的来源、授权和用途。

进入正式构建前必须满足：

- 来源清楚。
- 授权清楚。
- 是否可商用清楚。
- 是否允许二次修改清楚。
- 使用范围清楚。

禁止事项：

- 不提交授权不明素材。
- 不绕过验证码、付费墙、下载限制或站点反爬机制获取素材。
- 不把第三方账号、会员 cookie、API token 或下载凭据写入仓库。
- 不把“AI 生成”当作自动可商用结论；仍需记录模型 / 平台 / 生成时间 / 用途。

## 2. 当前素材盘点

## 3. 第一章视觉风格基线

本节为 T45 美术风格基线，不等于已引入正式素材。

核心关键词：

| 方向 | 关键词 | 用途 |
|---|---|---|
| 残榜 | 破裂卷轴、涂抹名籍、朱砂线、旧纸纤维 | 标题页、路线页、结算页主视觉 |
| 案前 | 木案、墨痕、笔锋、印章、案灯暗金 | 战斗 HUD、裁定页、奖励页 |
| 庙宇 | 香灰、旧庙梁柱、青铜铃、暗火 | 背景、事件页、休整页 |
| 器物 | 青铜、残玉、裂针、鞭节、镜面氧化 | 法宝栏、法宝奖励、反噬反馈 |
| 敌形 | 纸面、灰烬、兽影、旧役吏、空壳 | 普通敌人、精英、Boss 占位到正式资产 |

UI 基调：

| 模块 | 基调 | 避免 |
|---|---|---|
| 战斗页 | 桌案审判，信息密度高，关键反馈短促醒目 | 大面积二次元立绘、蓝紫仙侠渐变 |
| 路线页 | 榜面名籍节点，朱砂连线和烧痕区分节点 | 普通爬塔圆点地图观感 |
| 裁定页 | 残榜展开，三类裁定像三枚判印 | 杀 / 放二选一、敌魂收集暗示 |
| 法宝栏 | 器物悬挂或贴边陈列，强调认主 / 反噬状态 | 把法宝画成卡牌奖励本体 |
| 结算页 | 本局改榜记录，伏诛 / 归册 / 裁定留痕 | 只显示普通战绩数字 |
