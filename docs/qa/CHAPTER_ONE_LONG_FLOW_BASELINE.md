## 2. 多敌遭遇基线

- `enemySlots` 为新增兼容字段；旧 `enemyDefinitionId` 仍作为单敌 fallback 和主目标。
- 本轮只接 3 个多敌模板，不做召唤、多波次、站位规则或完整随机地图。
- 核心战斗支持目标选择、单个敌人形归零后的 `ENEMY_SETTLED` 记录、全体敌人归零后的 `VICTORY_SETTLED`。
- 最终战后奖励 / 裁定仍以主目标结算质量为准，避免支援杂祟串掉归册对象。

## 5. 存档与异常状态

- 当前存档版本升级到 v2。
- 读取兼容 v1 存档并归一化为 v2。
- 读取坏档、未知版本、非法 JSON 或浏览器 storage 抛错时，回退为无存档或默认设置。
- 写入设置 / 存档时若 storage 抛错，不中断页面流程。
- 清理存档会同时清理当前 v2 key 与旧 v1 key。
