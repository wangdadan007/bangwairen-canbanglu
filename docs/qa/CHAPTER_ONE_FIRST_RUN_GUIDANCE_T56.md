# T56 首局理解与关键术语引导补强记录

日期：2026-05-20

## 1. 本轮目标

让第一次玩的玩家不读 PRD，也能从 UI 里理解问名、归册、裁定、三坛和法宝为什么值得做；同时保持提示短、贴近当前状态，不做长篇教程墙。

## 2. 完成内容

- 新增首局上下文提示层，根据当前状态切换首战、第二战、第三战、战后奖励、裁定、朱批、商店、休整、三坛、法宝、反噬和 Boss 压力提示。
- 首战提示使用“查明真名（问名）”“真名已明（正名）”“归册”等白话 + 术语写法，明确伏诛和归册的收益差异。
- T75 后奖励提示已更新为：伏诛与归册卡牌奖励同池三选一；归册额外收益来自裁定、名破和机制解除。
- 裁定提示明确登簿、朱批、削籍三类选择的差异，不出现留魂、留名、敌魂卡或 Boss 助战。
- 商店 / 休整提示明确法宝是牌组外器物，不进入牌堆；休整会解释修形、删牌、朱批和反噬维护的取舍。
- Boss 提示聚焦来势、异动、问名、正名、法宝反噬和榜裂对收束质量的影响。
- 商店和休整的不可操作按钮补充 title 与可见原因，避免只看到灰按钮。
