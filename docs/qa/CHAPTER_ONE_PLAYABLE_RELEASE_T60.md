## 3. 当前可交付状态

第一章当前可以作为小范围外部试玩版本交付，前提是按 T59 的传播限制执行。

当前覆盖：

- 完整第一章路线：普通战、事件、商店、休整、精英、Boss 和第一章结算。
- 核心机制：破形、问名、正名、伏诛 / 归册、裁定、三坛、法宝三选一、反噬、己形、榜裂。
- 内容规模：44 张卡牌、12 个普通敌人单位、21 个普通遭遇模板、3 个精英、1 个 Boss、10 件法宝、9 个事件。
- 页面：标题、设置、路线、战斗、奖励、裁定、朱批、事件、商店、休整、失败结算和通关结算。
- 运行方式：浏览器 production build 与本地 preview；本地存档、设置和错误反馈记录可用。
