# T81 三角色非推荐路线真实拿牌观察与调平

日期：2026-05-24

## 目标

T81 先观察三角色各走一条非推荐路线时的完整局表现，并按用户最新指令，直接在 T81 内完成“非推荐路线生存与临诏首局节奏”调平。

记录项包括 Boss 前己形、Boss 后己形、战斗回合数、抓牌、删牌、朱批、法宝、墨 / 劫数 / 榜裂、临诏入场与触发次数。

## 观察口径

- 观察方式：临时 Vitest 自动观察脚本，调用当前 route / run / battle reducer 真实推进，不静态手填。
- 路线样本：莲烬走稳行休整线，衡简走归册事件商店线，照微走裂榜塞卷商店线。
- 出牌策略：偏向临诏、过牌、问名、封势和破形；不是最优人工策略，但能稳定暴露压力下限。
- 拿牌策略：每次战后奖励真实从当前 `rewardResolver` offer 中选择，优先临诏，其次问名 / 破形方向。
- 商店策略：调平后每个商店最多删 1 张牌，再考虑朱批服务、临诏牌或其他卡牌。
- 休整策略：调平后优先修形，再考虑朱批服务或删牌。

## T81 调整

- 将明烛临诏 `card_trace_name_slip` 与朱痕临诏 `card_red_ink_trial` 后移到 `stage_human_altar`，不再进入前三场教学奖励 / 商店窗口。
- `rewardResolver` 将 `linzhao` 纳入机制 tag 过滤，人坛 / 三坛解锁 keyword 补入 `linzhao`，避免只靠 `unlockStage` 放行临诏奖励。
- 商店朱痕临诏商品同步要求 `stage_human_altar`；商店删牌服务改为整局非重复购买，避免首局误删过多基础破形牌。
- 降低中后段连续掉形压力：瘟纸人、塞卷吏、逃名纸马、供桌残像、火部逃名者、斗部空壳、窃榜使均做小到中幅数值下调。
- 裂榜高压线在斗部空壳后新增 Boss 前休整节点 `route_node_boss_ante_rest`，给低己形或非推荐路线留下修形窗口。
- 照微奖励软倾向补入 `break_form` 权重，并降低纯坛位 / 天坛问名牌权重，避免洞察角色在真实拿牌中被推成低伤牌组。
