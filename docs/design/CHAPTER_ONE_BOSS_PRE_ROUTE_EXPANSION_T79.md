# T79 Boss 前 8-9 战路线与怪物内容设计与落地

日期：2026-05-24

## 1. 目标

将第一章进入窃榜使 Boss 前的每条实际路线扩为 8 到 9 场战斗。这里的“战斗”只计算普通战、精英战和其他会进入战斗页的遭遇；事件、商店、休整不计入战斗数，但仍计入路线节点和补给节奏。

本轮已完成内容设计与首版路线数据落地，优先复用当前 12 个普通 / 低成本敌人、3 个精英、1 个 Boss 和现有 21 个普通遭遇模板；如 QA 证明复用感过强，再另开任务补 1 到 2 个新遭遇模板，而不是先新增敌人单位。

## 2. 设计边界

- 保留前三场固定教学：纸面鬼、偷香鼠、铜铃夜巡。
- 保留第 4 节点后的三主线：稳行、归册、裂榜。
- 保留三角色全路线可走，不做角色锁路。
- 保留当前 Boss 窃榜使，不新增 Boss 池或第二章。
- 保留当前法宝节奏：角色固定起始法宝，第一章途中一次三选一，Boss 后一次三选一。
- 不新增生产依赖、正式素材或新核心机制。
- 新增战斗不应回头加重前三场教学，也不把普通战做成精英强度。

## 3. 当前缺口

T79 开始前，Boss 前每条路径战斗数偏短：

| 路线 | 当前 Boss 前战斗数 | 问题 |
|---|---:|---|
| 稳行：商店 / 休整支线 | 6 | 基准线偏短，法宝、朱批和事件收益尚未形成足够构筑轮廓。 |
| 归册：双精英支线 | 6 | 高压但短，连续精英后缺少普通战来消化构筑收益。 |
| 归册：事件商店支线 | 5 | 最短，归册线味道弱，Boss 前成长不足。 |
| 裂榜：事件休整支线 | 5 | 风险线反而短，劫数 / 榜裂压力和补偿都没有展开。 |
| 裂榜：塞卷商店支线 | 6 | 有高压节点，但 Boss 前污染 / 遮名 / 坛位考核不足。 |

目标是所有具体支线都到 8 场；若后续人工试玩认为第一章仍偏短，可只把高压支线扩到 9 场。

## 4. 总体节奏

每条路线按同一节拍扩展：

| 段落 | 战斗序号 | 功能 |
|---|---:|---|
| 固定教学 | 1-3 | 形 / 问名 / 来势 / 异动 / 朱批的基础教学。 |
| 主线入口 | 4 | 玩家选择稳行、归册或裂榜后的第一场路线身份战。 |
| 中段构筑 | 5-6 | 增加普通战或精英战，让奖励、朱批、法宝和墨开始成型。 |
| Boss 前考核 | 7-8 | 使用污染、遮名、召唤、地坛或支援组合，检查玩家是否准备好应对 Boss。 |
| Boss | Boss | 窃榜使综合考核。 |

事件、商店和休整放在战斗 4 到 7 之间，用来抵消高压支线，不直接替代战斗数。

## 5. 稳行线设计

稳行线作为基准路线，目标是 8 场战斗、1 个精英、1 个明确补给节点。它应该让新玩家能看见更多普通敌人和多敌组合，但不连续塞高压精英。

| 顺序 | 节点功能 | 建议遭遇池 | 设计目的 |
|---|---|---|---|
| 1 | 固定战 | 纸面鬼 | 教形、问名、正名。 |
| 2 | 固定战 | 偷香鼠 | 教来势 / 异动 / 断供。 |
| 3 | 固定战 | 铜铃夜巡 | 教封势与朱批入口。 |
| 4 | 主线入口 | 无灯庙祝 | 承接召唤异动和人坛。 |
| - | 补给二选一 | 纸钱小肆 / 旧庙案前 | 商店或休整，玩家自主补牌 / 修形 / 朱批。 |
| 5 | 中段普通 | 灰坛童子、铜铃夜巡再审、纸面鬼与无名纸祟 | 轻量复习地坛、来势和多敌。 |
| 6 | 精英 | 窃香吏 | 墨和断异动收益的核心考核。 |
| 7 | 支援普通 | 供桌残像护偷香鼠、偷香鼠与香灰虱、供桌残像再审 | 让玩家处理支援 / 偷香组合，不压成精英。 |
| 8 | Boss 前普通 | 瘟纸人、塞卷吏、瘟纸人再审 | 给 Boss 前一次污卷 / 净卷 / 墨资源检查。 |
| Boss | Boss | 窃榜使 | 稳行线 Boss 开场保持当前基础压力。 |

稳行线不需要新敌人。首版已新增 2 个普通战节点，并复用现有后段遭遇池抽样。

## 6. 归册线设计

归册线强调问名、正名、精英收益和遮名压力。它允许一条高压双精英支线，也允许一条事件商店支线，但两条都必须在 Boss 前达到 8 场战斗。

### 6.1 双精英支线

| 顺序 | 节点功能 | 建议遭遇池 | 设计目的 |
|---|---|---|---|
| 1-3 | 固定战 | 纸面鬼、偷香鼠、铜铃夜巡 | 固定教学。 |
| 4 | 归册入口精英 | 窃香吏 | 让玩家尽早看到精英归册价值。 |
| 5 | 第二精英 | 火部逃名者 | 强化遮名 / 削籍 / 榜裂压力。 |
| - | 补给 | 高案短息 | 连续精英后必须给休整。 |
| 6 | 名迹普通 | 逃名纸马、逃名纸马再审、供桌残像护偷香鼠 | 问名 / 护名 / 留墨的普通战复习。 |
| 7 | 坛位普通 | 灰坛童子、铜铃夜巡再审、无灯庙祝再审 | 让归册线也要处理坛位和来势。 |
| 8 | Boss 前普通 | 瘟纸人、塞卷吏、供桌残像再审 | 让 Boss 前有污染或支援压力。 |
| Boss | Boss | 窃榜使 | 归册倾向 Boss 开场偏遮名 / 遮势。 |

### 6.2 事件商店支线

| 顺序 | 节点功能 | 建议遭遇池 | 设计目的 |
|---|---|---|---|
| 1-3 | 固定战 | 纸面鬼、偷香鼠、铜铃夜巡 | 固定教学。 |
| 4 | 归册入口精英 | 窃香吏 | 保留归册线身份。 |
| - | 事件 | 中卷岔笔 | 给墨、法宝、榜裂或三坛事件。 |
| - | 补给 | 卷边市声 | 事件后接商店，避免事件分支过弱。 |
| 5 | 名迹普通 | 逃名纸马、逃名纸马再审、供桌残像护偷香鼠 | 先进入遮名压力。 |
| 6 | 坛位普通 | 灰坛童子、无灯庙祝再审、铜铃夜巡再审 | 补一次三坛 / 来势检查。 |
| 7 | 精英 | 火部逃名者 | 事件支线也必须面对归册线关键精英。 |
| 8 | Boss 前普通 | 瘟纸人、塞卷吏、供桌残像再审 | 给 Boss 前最后一次污染 / 支援准备。 |
| Boss | Boss | 窃榜使 | 归册倾向 Boss 开场偏遮名 / 遮势。 |

归册线复用火部逃名者，不新增第 4 个精英。两条支线的差别是火部逃名者出现位置不同：双精英支线更早更凶，事件商店支线更可规划。

## 7. 裂榜线设计

裂榜线强调劫数、榜裂、污卷和高压补给。它必须更凶，但不能只是多惩罚；事件、商店和休整都要服务“冒险换爆发”。

### 7.1 事件休整支线

| 顺序 | 节点功能 | 建议遭遇池 | 设计目的 |
|---|---|---|---|
| 1-3 | 固定战 | 纸面鬼、偷香鼠、铜铃夜巡 | 固定教学。 |
| 4 | 裂榜入口 | 断签客 | 劫数交易和风险记账入口。 |
| - | 事件 | 残榜空名 | 给卡、删卡、榜裂诱惑或朱批服务。 |
| - | 补给 | 裂榜残灯 | 事件后休整，抵消裂榜压力。 |
| 5 | 坛位普通 | 灰坛童子、铜铃夜巡再审、无灯庙祝再审 | 检查地坛和来势处理。 |
| 6 | 污染普通 | 瘟纸人、塞卷吏、瘟纸人再审 | 进入污卷压力。 |
| 7 | 遮名普通 | 逃名纸马、供桌残像护偷香鼠、供桌残像再审 | Boss 前准备护名 / 留墨。 |
| - | 事件 | 终审前残页 | Boss 前给净卷、缚名、法宝或榜裂选择。 |
| 8 | 终段精英 | 斗部空壳 | 天坛 / 法宝认主 / 坛位考核。 |
| Boss | Boss | 窃榜使 | 裂榜倾向 Boss 开场偏塞污卷。 |

### 7.2 塞卷商店支线

| 顺序 | 节点功能 | 建议遭遇池 | 设计目的 |
|---|---|---|---|
| 1-3 | 固定战 | 纸面鬼、偷香鼠、铜铃夜巡 | 固定教学。 |
| 4 | 裂榜入口 | 断签客 | 劫数交易。 |
| 5 | 高压普通 | 塞卷吏、塞卷吏再审、供桌残像再审 | 直接进入污染压力。 |
| - | 补给 | 碎页暗市 | 高压后给商店。 |
| 6 | 污染普通 | 瘟纸人、瘟纸人再审、塞卷吏再审 | 污卷二次考核。 |
| - | 事件 | 终审前残页 | Boss 前净卷 / 缚名 / 法宝事件。 |
| 7 | 遮名普通 | 逃名纸马、供桌残像护偷香鼠、香灰虱 | 检查遮名与支援。 |
| 8 | 终段精英 | 斗部空壳 | 天坛 / 法宝认主 / 坛位考核。 |
| Boss | Boss | 窃榜使 | 裂榜 / 高压倾向 Boss 开场偏塞污卷。 |

裂榜线第一版也保持 8 场。若后续要把某些路径扩到 9 场，优先给塞卷商店支线增加一场低形多敌战，而不是再加一个精英。

## 8. 怪物与遭遇池分配

第一版不新增敌人单位，只重排现有敌人与遭遇：

| 敌人 / 遭遇 | 新用途 |
|---|---|
| 灰坛童子 | 从“已有但路线存在感不足”提升为稳行、归册、裂榜中段坛位检查。 |
| 铜铃夜巡再审 | 作为低风险来势复习，适合和灰坛童子同池。 |
| 无灯庙祝再审 | 用作中段召唤复习，不放在 Boss 前最后一场。 |
| 供桌残像护偷香鼠 | 作为支援 + 资源压力组合，适合稳行 / 归册后段。 |
| 瘟纸人 / 塞卷吏 | 作为 Boss 前污卷检查，裂榜线出现更频繁。 |
| 逃名纸马 | 作为 Boss 前遮名检查，归册和裂榜线都应稳定可见。 |
| 火部逃名者 | 归册线关键精英，出现位置随支线变化。 |
| 斗部空壳 | 裂榜线终段精英，不提前进入稳行或归册线。 |

如后续需要新增遭遇模板，优先新增组合遭遇，而不是新增敌人单位：

| 候选遭遇 | 组合 | 适合路线 |
|---|---|---|
| 灰坛童子 + 铜铃夜巡 | 地坛检查 + 来势复习 | 稳行、归册、裂榜 |
| 瘟纸人 + 塞卷吏 | 慢性污染 + 直接污染 | 裂榜后段，只作为高压池候选 |
| 逃名纸马 + 香灰虱 | 遮名 + 低形偷香 | 归册 / 裂榜 Boss 前 |
| 无灯庙祝 + 无名纸祟 | 召唤压力复习 | 稳行中段 |

这些组合都复用现有敌人，不新增机制。

## 9. 经济与平衡注意点

路线从 5-6 场拉到 8 场后，玩家会多拿 2-3 次战后卡牌奖励，也会多触发法宝、登簿和朱批机会。第一版落地时暂不改奖励池和 Boss 数值，先通过敌人压力和路线补给观察真实手感；若 Boss 明显变弱，再从以下顺序调整：

1. 降低新增普通战的奖励质量权重，或让新增池更多出基础构筑补强而非高爆发牌。
2. 轻调 Boss 开场路线压力，而不是直接抬高 Boss 形。
3. 再评估 Boss 形或关键来势数值。

普通“残名入档”当前第一章本幕段维持 3 层额度。8 场 Boss 前战斗仍处在 7-8 场左右保持 3 层的口径；若某条路径后续扩到 9 场并且归册收益不足，再考虑是否将本幕段额度提高到 4 层。

## 10. 开发验收

T79 开发落地后，至少需要保留以下验证：

- 路线测试枚举所有具体支线，断言 Boss 前战斗数为 8 到 9。
- 三角色 × 三主线 pure-logic smoke 仍能完成第一章，且 pendingReward / pendingVerdict / pendingRedInk / pendingArtifactOffer 最终清空。
- 额外覆盖每条主线的非推荐角色支线：莲烬走稳行休整、衡简走归册事件商店、照微走裂榜塞卷商店。
- 法宝途中三选一仍能在完成 5 场遭遇左右触发，不依赖某个唯一精英遭遇。
- 路线倾向仍只做软提示：`steady`、`catalogue`、`fracture`、`supply`、`high_pressure` 不得变成角色限制。
- 浏览器人工试玩至少记录每个角色一条非推荐路线的 Boss 前己形、Boss 后己形、实际补给使用、污染 / 遮名 / 断异动压力。

后续若继续打磨 T79 手感，建议只做浏览器人工试玩与小幅数值 / 池子调整：用三角色各跑一条非推荐路线，记录 Boss 前己形、补给使用、污染 / 遮名 / 断异动压力，再决定是否微调新增节点 encounterPool、Boss 开场压力或补给位置。
